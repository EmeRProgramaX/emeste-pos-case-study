# EMESTÉ POS — Caso de Éxito

## Sistema POS local-híbrido para restaurante La Pica del Picaflor

**EMESTÉ POS** nace a partir de un proyecto real desarrollado para un restaurante en operación: **La Pica del Picaflor**.

El objetivo fue construir una solución práctica, rápida y confiable para gestionar la operación diaria del local, incluyendo mesas, comandas, caja, usuarios por rol y soporte técnico continuo.

> Este repositorio es un caso de éxito comercial. No expone código privado, datos sensibles ni información operativa confidencial del cliente.

## Problema detectado

El negocio necesitaba una herramienta simple, rápida y adaptada a su forma real de trabajar.

Los desafíos principales eran:

- Controlar mesas y comandas.
- Separar funciones por tipo de usuario.
- Mejorar la operación entre garzón, caja y administración.
- Mantener el sistema funcionando en red local.
- Reducir errores operativos.
- Contar con soporte técnico rápido y cercano.

---

## Solución implementada

EMESTÉ desarrolló un sistema POS local-híbrido adaptado al flujo real del restaurante.

La solución permite operar desde una red local, usando un computador principal como servidor y diferentes accesos para los usuarios del negocio.

### Roles principales

| Rol | Función |
|---|---|
| ADMIN | Gestión general, reportes, configuración, inventario y control operativo |
| CAJA | Cobros, cierre de caja, ventas y operación financiera del turno |
| GARZÓN | Gestión de mesas, comandas y atención operativa |
